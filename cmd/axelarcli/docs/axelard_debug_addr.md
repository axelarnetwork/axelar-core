## axelard debug addr

Convert an address between hex and bech32

### Synopsis

Convert an address between hex encoding and bech32.

Example:
$ <appd> debug addr cosmos1e0jnq2sun3dzjh8p2xq95kk0expwmd7shwjpfg

```
axelard debug addr [address] [flags]
```
