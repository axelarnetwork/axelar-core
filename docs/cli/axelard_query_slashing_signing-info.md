## axelard query slashing signing-info

Query a validator's signing information

### Synopsis

Query a validator's signing information, with a pubkey ('<appd> comet show-validator') or a validator consensus address

```
axelard query slashing signing-info [validator-conspub/address] [flags]
```

### Examples

```
axelard query slashing signing-info '{"@type":"/cosmos.crypto.ed25519.PubKey","key":"OauFcTKbN5Lx3fJL689cikXBqe+hcp6Y+x0rYUdR9Jk="}'
```

  -h, --help               help for signing-info
